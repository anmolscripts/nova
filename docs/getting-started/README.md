# Welcome to Learn Nova

Learn Nova is the **official beginner-friendly guide** for Nova Framework.

Unlike traditional documentation, this guide teaches Nova by building a **real-world application** from scratch.

Instead of reading isolated API examples, you'll progressively build an application while learning every major feature of the framework.

Whether you're coming from **Plain PHP**, **Laravel**, **CodeIgniter**, **ASP.NET**, or **Next.js**, this guide is designed to help you become productive with Nova as quickly as possible.

# Before You Continue

Nova is not a traditional MVC framework.

Instead of organizing code around Controllers, Models and Views...

Nova organizes applications around **Features**.

```

Users

│

├── page.php

├── page.latte

├── page.ts

├── page.scss

├── actions/

├── components/

└── middleware.php

```

Everything related to a feature lives together.

This makes applications easier to understand, maintain and scale.
